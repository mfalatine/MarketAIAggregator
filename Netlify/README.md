# Market AI Aggregator — Netlify

This application is exclusively for direct cloud APIs. It contains no
subscription CLI execution choice and does not show CLI authentication,
executable paths, subscription models, or CLI administration.

- Open `index.html` directly or deploy this folder to Netlify.
- Anthropic, Gemini, and OpenAI all use API keys configured in the browser.

Configure providers under **Settings → Connections** and choose the API model
under **Settings → Briefing profiles**. Provider keys are stored in this
browser and excluded from normal exports.

Development verification:

```powershell
pnpm install
pnpm check
```
